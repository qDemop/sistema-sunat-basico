# Glossary

| Term | Definition |
|---|---|
| AFP | Private pension regime. The applied employee deduction percentage comes from the single Active effective-dated pension configuration version and is snapshotted in payroll detail. |
| ONP | National pension regime. The applied employee deduction percentage comes from the single Active effective-dated pension configuration version and is snapshotted in payroll detail. |
| CTS | Compensacion por Tiempo de Servicios, calculated semiannually using salary and proportional gratification. |
| Gratification | Legal benefit paid in July and December, represented in the PDFs as 1/6 of gross salary. |
| IGV | Peruvian value-added tax. The PDFs use 18%. |
| SUNAT | Peruvian tax authority. The system must support tax books and current tax rules. |
| Libro de Compras | SUNAT purchase book generated from purchase vouchers. |
| Libro de Ventas | SUNAT sales book generated from sales vouchers. |
| Comprobante | Accounting voucher: invoice, sales receipt, credit note, debit note, remittance guide, or receipt for fees. |
| RBAC | Role-Based Access Control. Access is based on predefined user roles. |
| JWT | JSON Web Token used for authenticated API calls. |
| BCNF | Boyce-Codd Normal Form used as the target database normalization level. |
| CQRS | Command Query Responsibility Segregation: commands mutate state, queries read state. |
